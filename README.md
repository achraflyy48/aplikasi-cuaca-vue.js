# 🌤️ Aplikasi Cuaca Vue.js

Sebuah aplikasi pemantau cuaca interaktif yang dibangun menggunakan **Vue.js 3** dan **Vite**. Project ini dikembangkan untuk mengasah logika pemrograman *front-end*, pengelolaan *state*, serta integrasi RESTful API secara asinkron.

## ✨ Fitur Utama

* **Pencarian Real-time:** Pengguna dapat mencari kondisi cuaca berdasarkan nama kota.
* **Data Akurat:** Menampilkan informasi suhu, kelembaban, dan kondisi cuaca terkini.
* **Modern & Responsif:** Antarmuka pengguna yang bersih dan dapat diakses dengan baik melalui berbagai ukuran layar.
* **Optimalisasi Performa:** Dibangun menggunakan Vite untuk *Hot Module Replacement* (HMR) yang super cepat dan *build* yang ringan.

## 🚀 Panduan Instalasi (Menjalankan di Local)

Untuk menjalankan project ini di komputermu, ikuti langkah-langkah berikut:

1. **Clone repository ini**
   ```bash
   git clone [https://github.com/](https://github.com/)[USERNAME_GITHUB_KAMU]/aplikasi-cuaca-vuejs.git

2. **Masuk ke direktori project**
   ```bash
   cd aplikasi-cuaca-vuejs

3. **Install dependensi**
   ```bash
   npm install

4. **Konfigurasi Environment Variable**
   <ul>
     <li>Buat file .env di root direktori.</li>
     <li>Tambahkan API Key kamu (misalnya dari OpenWeatherMap):</li>
   </ul>
```bash
VITE_WEATHER_API_KEY=masukkan_api_key_kamu_disini
```

5. **Jalankan development server**
   ```bash
   npm run dev
